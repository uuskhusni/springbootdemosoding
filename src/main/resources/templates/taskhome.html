Welcome to Spring Boot MVC CRUD

Dear All, this site develop using spring boot MVC, MySQL as Database
This is show how to CRUD(Create, Delete, Update) data. This app i use Constructor Base Injection with flow Like this :

  1. Create Interface Class for injection, the advantage is more flexibel and best practice
  2. Create Implementation Class for Interface that we create before. this class like DAO(Data Access Object)
  3. Create Class to receive all injection result we call this class is controller
